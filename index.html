Tensorpac¶

Tensorpac is an open-source Python toolbox for computing Phase-Amplitude Coupling (PAC) using tensors and parallel computing. This software provides a modular implementation which allows one to combine existing methods for measuring PAC and chance distribution. To get started, see our examples.

Installation:¶

Tensorpac is based on NumPy (>=1.12), SciPy and uses Joblib for parallel computing. To install Tensorpac, just open a terminal and run :

pip install tensorpac
