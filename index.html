Tensorpac¶

Tensorpac is an open-source Python toolbox for computing Phase-Amplitude Coupling (PAC) using tensors and parallel computing. We also designed a modular implementation which let you combine existing methods for measuring Pac and chance distribution. We provide a set of examples.

Installation:¶

Tensorpac is based on NumPy (>=1.12), SciPy and use Joblib for parallel computing. For the installation, in a terminal run :

pip install tensorpac
